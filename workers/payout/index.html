This URL is not your website

You are viewing the payout Worker project folder. It only contains Cloudflare Worker code (wrangler.toml, src/). It must not be used as the static site root on IPFS, Netlify, 4EVERLAND, or GitHub Pages.

If the browser showed raw wrangler.toml text: your host’s publish / output / root directory is set to workers/payout. Change it to the repository root (where the hub index.html lives) or to yabbai/ if you only want the main app.

Deploy this folder only with Wrangler:

cd workers/payout
npm install
npx wrangler secret put TREASURY_SECRET_KEY
npx wrangler deploy

TREASURY_SECRET_KEY must be the keypair for the same on-chain address the static app shows (meta / window.YABBAI_TREASURY_RECEIVER if set, otherwise DEFAULT_TREASURY in yabbai/index.html). Optionally set plain var YABBAI_TREASURY_RECEIVER in wrangler.toml to that pubkey so GET treasuryPubkey / treasuryMatchesEnv can verify a match.

Then set window.YABBAI_PAYOUT_API in yabbai/mission-config.js to the Worker HTTPS URL.
