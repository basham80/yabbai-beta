Connect wallet
Connect lets this site read your public key and query SOL balance from the Solana RPC (the network), like a block explorer. An optional Sign Message step proves you control the address for login — it is off-chain and does not move funds. Sending SOL or tokens still requires you to approve each transaction in Phantom; Solana does not grant one-signature “full transfer access.”
